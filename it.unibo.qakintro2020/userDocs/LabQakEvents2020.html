Rules

  • The event emitted by a QActor is propagated to ALL the other actors, even when they are callers of ExternalActor
  • The event that reaches a Context is propagated ONLY to the actors internal to that Context


Legenda for the following figures
A could use R as an ExternalQActor
  • e0 (sent by C to R) is propagated ONLY to R

  • e1 (emitted by R) is propagated to A

  • e2 (emitted by A ) is propagated to R

  • e2 is not propagated to C
A and B could use R as an ExternalQActor
  • e1 (sent by C to R) is propagated to A and to B

  • e2 (emitted by A ) is propagated to R and A but not to C

  • e2 (emitted by B ) is propagated to R and B but not to C
